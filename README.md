# 🛒 GroceryHub – Modern Grocery eCommerce Web App
https://groceryhub-full-django-working-e-comerce.onrender.com/
GroceryHub is a full-featured grocery eCommerce web application built with Django and JavaScript.  
It is designed for supermarkets, mini-marts, and local grocery businesses to sell products online efficiently.

---

## 🚀 Features

### 🛍️ Shopping Experience
- Full-featured shopping cart
- Product search and category filtering
- Product ratings and reviews
- Pagination for large product lists
- Featured products carousel

### 💳 Checkout & Payments
- Secure checkout process
- PayPal integration
- Credit/Debit card support
- Shipping and order tracking system

### 👤 User System
- User registration & login
- User profile with order history

### 🛠️ Admin Dashboard
- Manage products, users, and orders
- Mark orders as delivered
- Add unlimited products and categories
- Blog and contact page management

---

## 🧰 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (default)
- **Payments:** PayPal API

## ⚙️ Installation Guide (Windows)

1. Clone the repository:
```bash
git clone [https://github.com/Lixxontech001/GroceryHub/
cd groceryhub

```
Create virtual environment
```

py -m venv venv
```


Activate environment:
```
.\venv\Scripts\activate

```
Install dependencies:
```bash
pip install -r requirements.txt

```
Run server:
```bash
py manage.py runserver

```
Open in browser:
```bash
http://localhost:8000/

```


🔐 Admin Access


Create admin user:

```

py manage.py createsuperuser


```
Access admin panel:

```bash
http://localhost:8000/admin/

```
